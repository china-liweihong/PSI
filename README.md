PSI的开源协议
-------------
>PSI的开源协议为GPL v3

>如果您有Sencha Ext JS的商业许可（参见： http://www.sencha.com/products/extjs/licensing/ ），那么PSI的开源协议为Apache License v2。

>GPL v3协议下，您必须保持PSI代码的开源。 
>Apache License协议下，您可以闭源并私有化PSI的代码，作为您自己的商业产品来销售。
